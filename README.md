# ChefNote 🍳

An AI-powered recipe manager web application built using PHP, MySQL, Bootstrap 5, and the Groq AI API (Llama 3.3 70B).

## Features
* **User Authentication:** Secure registration and login portals with password hashing.
* **Kitchen Counter:** Custom recipe building with automatic text formatting.
* **Mood Lab:** Track and match recipes dynamically based on culinary preferences.
* **AI Cooking Assistant:** Interactive chat interface to request custom recipe changes, cooking techniques, or ingredient substitutions.

## Tech Stack
* **Backend:** PHP, MySQL
* **Frontend:** Bootstrap 5, Custom CSS, Vanilla JavaScript
* **AI Engine:** Groq API (Llama 3.3 70B Versatile)
